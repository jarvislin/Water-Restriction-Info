限水資訊 - Android
==========
因為水庫枯旱的關係，最近會有一些限水的政策，

這個App抓取了自來水公司的公告及水庫資訊，

可透過公告得知停水區域及日期，

亦可瀏覽水庫狀況。

下載
---------
已下架，請自行安裝。
